## Core Skills

* **Graphics:** WebGL2, GLSL, real-time rendering.
* **Web:** JS, Lit, React, WebAssembly, Emscripten, Coffeescript.
* **Systems:** C, C++, low-level programming.
* **Control & Embedded:** Beckhoff PLC integration, hardware-linked UI.
